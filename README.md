# Studiator - Management Dashboard

## Author: Aklima Akter Mouri

## Overview
**Studiator** is a full-stack web application designed to provide a streamlined management interface. This project focuses on implementing secure user authentication and a dynamic administrative dashboard, demonstrating core competencies in backend logic and responsive frontend design.

## Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Grid and Flexbox layouts)
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** XAMPP / WAMP

## Key Features
* **Secure Authentication:** Fully functional and secure Login and Sign-up modules.
* **Profile Management:** Dynamic user profile pages allowing for personalized data display and updates.
* **Administrative Dashboard:** A centralized management interface with a professional, responsive aesthetic.
* **Database Persistence:** Integrated MySQL backend to handle user records and system data efficiently.

## Setup & Installation
1. **Database Setup:** 
   * Open phpMyAdmin and create a new database named `studiator`.
   * Import the provided `studiator.sql` file.
2. **Configuration:** 
   * Open the database connection file (e.g., `config.php`) and update the database name, username, and password to match your local setup.
3. **Run Project:**
   * Move the project folder to your local server directory (`htdocs` or `www`).
   * Access the project in your browser at `http://localhost/Studiator`.
